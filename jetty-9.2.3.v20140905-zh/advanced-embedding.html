这节提供一个教程向你展示如何快速开发一个针对 Jetty API 的嵌入式的代码.

下载 Jars

Jetty 被分解为许多 jars 和依赖,通过选择 jars 的最小集合实现最小的占用.通常最好使用类似 Maven 的方式管理 jars,不过这个教程使用一个包含所有的 Jetty classes 的 jar.你可以手动下载聚合的 jetty-all.jar 和 servlet-api.jar,使用 curl 或一个浏览器.

注意

中央 maven 库已经开始积强制地拒绝接受/拒绝访问存储库从 wget 命令行工具中(由于这个工具被一些组滥用).中央 maven 库的管理员已经规定推荐的命令行下载工具,就是现在的 curl.

如下使用 curl:

> mkdir Demo
> cd Demo
> curl -o jetty-all.jar http://central.maven.org/maven2/org/eclipse/jetty/aggregate/jetty-all/9.2.3.v20140905/jetty-all-9.2.3.v20140905.jar
> curl -o servlet-api.jar http://central.maven.org/maven2/javax/servlet/javax.servlet-api/3.1.0/javax.servlet-api-3.1.0.jar

编写 HelloWorld 示例

The 嵌入 Jetty 一节包含了许多针对 Jetty API 编写的示例. 这个教程使用了一个简单的 HelloHandler,有一个 main 方法运行服务器.你也可也 href="http://git.eclipse.org/c/jetty/org.eclipse.jetty.project.git/plain/examples/embedded/src/main/java/HelloWorld.java" target="_top">下载 或在一个编辑器中创建一个使用如下内容的 HelloWorld.java.

编译 HelloWord 示例

下面的命令行编译 HelloWorld 类:

javac -cp servlet-api.jar:jetty-all.jar HelloWorld.java

运行 Handler 和 Server

下面的命令行运行 HelloWorld 示例:

java -cp .:servlet-api.jar:jetty-all.jar HelloWorld

你可以将你的浏览器指向 http://localhost:8080 查看你的 hello world 页面.

下一步

  • 探索完整的 jetty javadoc

  • 考虑使用 Jetty 和 Maven 管理你的 Jars 和依赖.
