基于启动配置管理 XML

当你看到命令行上用于 Jetty 启动的 XML 文件,它们总是 Jetty IoC 配置机制的一部分.

在内部,Jetty 使用这些你想使用的特性的 XML 文件来构建 Jetty.

存在于 Jetty 中的 module 机制决定了 XML 文件的加载顺序.

Jetty Base 和 Jetty Home 解决逻辑也同样适用,允许你通过一个模块覆盖一个 XML 文件声明通过使用你的 XML 通过在你的 ${jetty.base} 目录位置有相同命名的 XML.
