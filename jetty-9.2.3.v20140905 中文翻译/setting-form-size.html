Jetty 限制了从一个浏览器或其它客户端传回到服务器的数据量.这有助于保护服务器针对恶意客户端发送大量的数据而拒绝服务攻击. Jetty 的默认最大大小许可是 200000 bytes.你可以针对一个特定的 webapp, 一个服务器上的所有 Apps, 或 同一个 JVM 中的所有的 Apps, 修改这个默认值,

你可以在 webapp 的外部的 context XML 部署描述符,或者在 webapp 的 WEB-INF 目录中的 jetty-web.xml 完成这个.
